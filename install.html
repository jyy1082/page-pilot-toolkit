page-pilot-toolkit

Drag the button below to your bookmarks bar. Click it on any site to open a record/run panel — no install, no extension.

▶ PagePilot

Drag this to your bookmarks bar (don't just click it here — dragging is how you install it).

How to use it

  • Go to any website, click the PagePilot bookmark.
  • A small panel appears in the corner. Press Start recording, then use the page normally — type, click, select, whatever the task needs.
  • Press Stop. The recorded steps show up as JSON in the box.
  • Press Run to play them back right there, or Copy to take the JSON somewhere else (paste it into cursor.run(steps) in your own code).
  • You can also paste your own hand-written steps array into the box and press Run directly — recording isn't required.

What it won't do

  • Doesn't save anything between visits. Closing the tab (or the panel) loses whatever's in the box — copy it somewhere first if you want to keep it.
  • Never records password fields, on any site, no matter what — this is a hard rule in the recorder itself, not something this panel adds on top.
  • Can't run on every site. Some sites set a Content-Security-Policy that blocks loading external scripts entirely — the bookmarklet will show an alert if that happens. That's the site's own security setting; there's no way around it from a bookmarklet (a browser extension has more privilege and could work around this, a bookmarklet can't).
A page you run this on gets exactly the same access your own browser session already has to that page — same as any bookmarklet or user script. Don't run it on anything you wouldn't otherwise paste random JavaScript into.
